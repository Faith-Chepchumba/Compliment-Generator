# Compliment Generator

## Overview

A Python program that generates a random compliment each time you press Enter.

## Features

- Press Enter to receive a compliment.
- Randomly selects from a list of compliments.

## Technologies Used

- Python

## How to Run

1. Run `compliment_generator.py` using Python: `python compliment_generator.py`.

2. Press Enter to generate a random compliment.

3. Type 'quit' to exit the program.
